# CyberVerse AI

AI 网络安全学习平台 - 赛博朋克风格

## 功能

- AI 安全助手 (Coze LLM 集成)
- 学习中心 (Web安全/渗透测试/密码学/网络攻防/逆向工程/AI安全)
- 漏洞靶场 (DVWA/XSS/SSRF/FileUpload/CmdExec/CVE)
- 工具中心 (Port Scanner/GitHub Analyzer/Packet Analyzer/Subdomain Finder)

## 技术栈

- Next.js 16 (App Router)
- React 19
- TypeScript 5
- Tailwind CSS 4
- shadcn/ui

## 快速开始

```bash
pnpm install
pnpm dev
```

访问 http://localhost:5000
